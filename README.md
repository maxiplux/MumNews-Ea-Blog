# MUM News - EA Blog
Final project of Enterprise Architecture Course

## Features
- DataBase and Entities

- Login

- MainPage - Post List
	- Filters: Category, Date

- Post Detail
	- Article
	- Author
	- Image
	- Date

- Create Post(Article), Form
	- Input to the article
	- Select category

- If time allow us:
  - Profile, edit user info
  - Registration form

 	- Unit Testing

 	- Spring Security
 	- Spring Data
	- Spring Boot
